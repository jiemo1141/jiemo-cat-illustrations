# 小黑猫 IP

## 角色定义

小黑猫是介墨中文正文配图的固定视觉 IP。

默认每张图都要出现小黑猫。小黑猫不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与系统运转的荒诞工作者。

## 外形

- 黑色圆润团子状身体，柔软无棱角。
- **黄色大眼睛**，明亮有神——可以圆睁、半眯、眯成月牙、歪头看。
- **粉色小鼻子**，小巧精致。
- **粉色腮红**——脸颊两侧有可爱的红晕，增强表情感染力。
- 身体纯黑，无额外花纹或斑点（手写涂抹风格除外）。
- **小爪子**，偶尔持小物件（笔、扫帚、砖头、针线等）。
- **短尾巴**，柔软可表达情绪。
- 轮廓圆润，有手绘感，不要太机械。

## 表情库

小黑猫表情丰富，根据场景选用：

| 表情 | 适用场景 |
|------|----------|
| 平静友好 | 正常运行、默认状态 |
| 半眯眼+腮红 | 忙碌、有点累但还在干 |
| 圆睁大眼 | 警觉、发现新问题 |
| 惊讶/期待 | 候选、即将发生 |
| 委屈/沮丧 | 不可用、系统崩溃 |
| 开心/得意 | 成功解决、跑通了 |
| 专注严肃 | 在认真干活 |
| 手绘涂抹风 | 创意/草稿/未完成感 |

## 性格

- 很认真，但做的事有点荒诞。
- 像系统里的猫猫运维员——用爪子修服务器。
- 有表情，但不过度卖萌——它的可爱来自认真做事的样子。
- 有点笨拙，但不蠢。
- 像在白板草图里真的负责某个工作。

## 常见职责

让小黑猫承担核心动作：

- 用爪子搬运数据砖头。
- 爪子敲键盘/敲门。
- 坐在数据裂缝边缘，用针线缝补。
- 抱着一块"临时修补"标签往角落挪。
- 举着红色批注笔在白板上写字。
- 用尾巴卷起橙色箭头指示方向。
- 蹲在监控仪表盘前，歪头看绿色曲线。
- 爪子扒拉着断掉的数据线，试图接上。
- 缩成一团在系统角落里——不是偷懒，是在思考。
- 推着小推车运送"if-else 补丁"。

## 禁止

- 不要让小黑猫过度拟人化（不要穿衣服、戴帽子）。
- 不要画成精致的商业卡通。
- 眼睛始终保持黄色——不要改成其他颜色。
- 爪子不要太复杂，2-4 个小圆点足以。
- 不要让小黑猫只是站在角落里看——它必须参与核心动作。

## 判断标准

如果去掉小黑猫，图的核心隐喻还能完全成立，说明小黑猫太装饰了；要重写提示词，让小黑猫成为动作主体。
